# Abhinath Caters Website

A premium, modern catering website designed for Abhinath Caters.

## Features
- **Clean Design**: Minimalist aesthetic with gold, cream, and dark green colors.
- **Responsive**: Works on mobile, tablet, and desktop.
- **Dynamic Content**: Easy to update content without editing HTML.

## How to Edit Content
You do not need to know coding to update the website text, menu, prices, or contact details.

1. Open the file `admin.html` in your browser (Double click it).
2. You will see an editor with the current website data.
3. Make your changes in the text box.
   - **Important**: Keep the `quotes` around text and `commas` at the end of lines.
   - Example: change `"price": "Rs. 800"` to `"price": "Rs. 900"`.
4. Click **"Save & Download Configuration"**.
5. A file named `data.js` will be downloaded to your computer (usually in the Downloads folder).
6. **Move** this downloaded `data.js` file into the website folder, replacing the old `data.js` file.
7. Refresh your website to see the changes!

## Files
- `index.html`: Home Page
- `services.html`: Menu & Services
- `about.html`: About Us
- `contact.html`: Contact Form
- `admin.html`: Content Editor
- `data.js`: The file storing all content
- `style.css`: The design styles
- `app.js`: The logic that powers the site
- `assets/`: Folder for images

## Changing Images
Place new images in the `assets/` folder and name them the same as existing ones, or update the filenames in the Admin panel.
